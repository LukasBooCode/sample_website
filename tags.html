Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
blockquote blockquote everything inside is indented so it stands out compared to other content on a page, which is a style often applied to quotes
div divider no effect on its own, holds elements, so style, id, class or whatever can be applied
header header holds title and/or main topic of a page for SEO and screen-reader purposes

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make strong text <strong>at least a billion people</strong> at least a billion people
a anchor make hyperlink <a href="https:learnenough.com/">Learn Enough</a> Learn Enough
img image include an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
code code make text display in monospace font <code>x = 10;</code> x = 10;
span span element without any effects on its own, used to single out stretches of text to style, or do something else to, later <span style="color: red;">DAH GREAT</span> DD BREASTS DAH GREAT DD BREASTS
br line break inserts a line break into text Line 1<br>Line 2 Line 1
Line 2

Document Tags

Tag Name Purpose Example
html hypertext markup language tells browser we're using html <html><head>...</head><body>...</body></html>
head header contain data not rendered on webpage, but for use in how the webpage is rendered <head><title></title><meta charset="utf-8"></head>
body body everything inside this tag is what is rendered on the page <body><p>yada yada</p></body>
title title title that shows on tab <title>Learn Enough To Be Dangerous</title>
meta metadata meta content intended for information about the webpage content <meta charset="utf-8">
